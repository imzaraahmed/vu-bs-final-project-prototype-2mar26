
# Job Application Tracker - Prototype

Back-end (Node.js + Express)
Front-end (React + Shadcn/ui)
Database (MySQL)

## Root folders and files:
Backend
Frontend
Database.sql
README.md

## Backend Setup:
cd backend
npm install
create .env (add db credentials)
Start Server:
node server.js

## Frontend Setup:
cd backend
npm install
Start Server:
npm run dev
